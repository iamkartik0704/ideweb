Context & Token Lifecycle Optimization

  • Context Compression & Token Optimization: To aggressively reduce API overhead and bypass strict LLM context limits, the engine avoids embedding raw, multi-megabyte codebase text. Instead, it extracts key file references, mappings, and high-level abstract syntax trees (AST), translating them into ultra-dense, token-optimized summaries fed directly into the AI Agent panel.
  • Silently Weaved Context Assembly: Never forwards raw user prompts directly to the LLM. The system silently weaves a master prompt packaging the underlying OS environment, active filename, language ID, and the text buffer immediately surrounding the cursor location.

Advanced AI Debugging: "Fix with AI"

  • The Interactive AI Debugger Panel: A modern, visual debugging workflow triggered when runtime crashes or syntax errors occur. Selecting "Fix with AI" initializes a specialized debugging terminal overlay adjacent to your active file.
  • Diff Visualization & Auto-Apply: The agent automatically inspects the code fault, generates an optimized solution, and highlights the changes using a side-by-side or inline color-coded Git diff. Developers can instantly click Auto-Apply to securely patch the active Monaco text buffer.
  • Session History Management: Tracks your debugging journey step-by-step. The AI Debugger maintains a robust local state history, allowing you to review previous fix iterations, compare multiple AI suggestions, and roll back code to any stable state seamlessly if the agent introduces side effects.

AI Orchestration & Streaming Optimization

  • Non-Blocking IPC Streams: Feeds generated tokens directly into the frontend using chunked streams over IPC, completely preventing UI thread freezes during massive code blocks.
  • Token Debouncing & Batching: Accumulates incoming AI tokens into a background buffer and flushes them to the Monaco text model exactly every 16ms (using requestAnimationFrame). This strictly matches a smooth 60fps refresh rate, ensuring scrolling remains highly responsive while the AI "types".
  • Stream Recovery: Features built-in fault tolerance for AI code generation streams. If a network drop occurs mid-generation, the stream pipeline gracefully pauses and handles automated reconnection attempts without corrupting or breaking the active Monaco text buffer.
